## Fleet Timeline

The Fleet Timeline shows aircraft availability across time, making it easy to spot scheduling conflicts or find available aircraft.

### Switching Views

Toggle between **Calendar** and **Fleet Timeline** using the buttons above the main view.

### Timeline Features

- Each row represents one aircraft in your fleet
- Colored blocks show when aircraft are scheduled
- Gaps indicate availability
- Hover over blocks for quick details
