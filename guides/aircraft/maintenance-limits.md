# Maintenance Limits

Track hours and cycles against maintenance limits on each aircraft. Requires **Fleet & Crew Limits → Maintenance tracking** enabled in **Settings → Modules**.

***

## Step 1: Enable maintenance tracking

1. Go to **Settings → Modules**
2. Enable **Maintenance tracking** (Fleet & Crew Limits)
3. Save — the **Maintenance** tab appears on each aircraft

## Step 2: Open the Maintenance tab

1. Click **Aircraft** in the sidebar
2. Open an aircraft
3. Select the **Maintenance** tab

If tracking is not enabled, the tab shows setup instructions instead of limits.

## Step 3: Set the baseline

On first use:

1. Enter current totals (hours, cycles, landings as applicable)
2. Set the **as-of date** for the baseline
3. Save the baseline — future flights increment from this point

## Step 4: Add maintenance items

1. Click **Add maintenance item**
2. Enter the item name, limit type (hours, cycles, calendar), and threshold
3. Set warning thresholds (amber/red) if desired
4. Save

Items appear in the running totals table with current consumption vs limit.

## Step 5: Sign off completed maintenance

When maintenance is performed:

1. Open the item
2. Click **Sign off** and record the work completed
3. The item resets or rolls forward per your configuration

## Fleet-wide view

Open **Fleet Maintenance** from the Aircraft menu to see all aircraft with items approaching limits.
