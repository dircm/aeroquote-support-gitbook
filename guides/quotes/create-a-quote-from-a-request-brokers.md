# Create a Quote from a Request (Brokers)

Brokers use the same request → quote flow as operators, but work within broker permissions and external-operator aircraft where applicable.

## Prerequisites

* A broker account with quote permissions
* An existing [Request](../requests/creating-a-request-brokers.md)

***

## Step 1: Open the request

1. Click **Requests** in the sidebar
2. Open the request you want to convert, or create a new broker request first

## Step 2: Select aircraft and generate

1. Review the flight legs and cost estimates on the request
2. Click the aircraft option to include in the quote
3. Click **Generate Quote**

## Step 3: Complete the Quote Builder

1. Review **Quote Details** — customer and reference information
2. Review **Charter Flights** — legs, dates, airports
3. Confirm **Aircraft** selection (including external-operator aircraft if applicable)
4. Check the **Quote Summary** and click **Create Quote**

## Step 4: Finalise the quote

1. The quote opens on the detail page
2. Adjust pricing on the **Options** step if needed — see [Edit the Price of a Quote option](edit-the-price-of-a-quote.md)
3. Build the customer document and send when ready — see [Creating a Quote](creating-a-quote.md)
