# Create a Quote from a Request (Operators)

Creating quotes from requests is the fastest way to build a proposal — flight details and estimates are already on the request.

## Prerequisites

* At least one aircraft configured with performance and costings
* An existing [Request](../requests/creating-a-request-operators.md), or create one as part of this flow

***

## Step 1: Open the request

1. Click **Requests** in the sidebar
2. Open an existing request, or click **Add New Request** to create one
3. Ensure the request has flight legs and at least one aircraft estimate selected

## Step 2: Generate the quote

1. On the request page, click the aircraft row you want to include in the quote
2. Click **Generate Quote**
3. The **Quote Builder** opens with the request's flights pre-filled

## Step 3: Review the Quote Builder

1. **Quote Details** — confirm customer, contact, and quote metadata
2. **Charter Flights** — review legs, dates, and airports; adjust if the schedule changed since the request
3. **Aircraft** — confirm the selected aircraft and options
4. **Quote Summary** — review totals before creating

## Step 4: Create and open the quote

1. Complete the builder steps and click **Create Quote**
2. AeroQuote opens the new quote on the detail page
3. Continue with [Creating a Quote](creating-a-quote.md) (document builder, pricing, send) as needed

{% hint style="info" %}
You can also create quotes from scratch (without a request) using **Create Quote** on the Quotes page. See [Creating a Quote](creating-a-quote.md).
{% endhint %}
