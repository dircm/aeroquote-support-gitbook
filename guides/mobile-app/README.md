# 📱 Mobile App

Available on the [App Store](https://apps.apple.com/au/app/aeroquote/id6467820969) and [Google Play.](https://play.google.com/store/apps/details?id=com.aeroquote.aeroquotecrew\&pcampaignid=web_share)

The AeroQuote Crew mobile app gives your team access to bookings, flight details, passenger check-in, and real-time notifications — all from their phone.

Designed for **pilots, crew, and operations staff** who need quick access in the field without logging into the full web application.

## Key Features

* **Dashboard** — At-a-glance view of your next flight, active bookings, and fleet status
* **Bookings & Flights** — Browse all bookings, view flight legs with departure/arrival times, routes, and crew
* **Passenger Management** — View contact details, edit weights, add and remove passengers on flights
* **Passenger Check-In** — Check in passengers and cargo via toggle or QR code scanner
* **Flight Status** — Record block off/on, departure/arrival times; receive automatic notifications from FlightRadar24
* **Log Position & Update ETA** — Report GPS position and update estimated arrival during flights
* **Calendar Sync** — Sync flight schedules to your device calendar
* **Booking Sharing** — Share the customer booking link
* **Aircraft & Fleet** — View fleet aircraft you have been assigned to
* **Offline Support** — All operations work without internet; data syncs automatically when connectivity returns
* **Push Notifications** — Receive real-time alerts for booking updates
* **Help & Getting Started** — In-app help guide with swipe-through pages on first launch

## Supported Platforms

* **iOS** — iPhone (iOS 15.0+)
* **Android** — Android 5.0+ (API 21)

## Getting Started

1. Download the AeroQuote Crew app from on the [App Store](https://apps.apple.com/au/app/aeroquote/id6467820969) or [Google Play.](https://play.google.com/store/apps/details?id=com.aeroquote.aeroquotecrew\&pcampaignid=web_share)
2. Sign in with your existing AeroQuote account credentials
3. Your bookings and fleet data will sync automatically

{% hint style="info" %}
You use the same email and password as the AeroQuote web application. There is no separate registration for the mobile app.
{% endhint %}
