# Create a Booking from a Quote

You can create a booking directly from any accepted quote. This pre-fills the Booking Builder with the quote's contact, flights, and aircraft details.

### Steps

1. Open an existing [Quote](../quotes/)
2. Click the **Generate a Booking** button on the quote's detail page
3. The [Booking Builder](booking-builder.md) opens with the quote details pre-filled
4. Review and adjust the details — dates, aircraft, and passenger information may need updating
5. Follow the builder steps through to **Create Booking**

Your new booking opens automatically on the booking detail page.

{% hint style="info" %}
You can also create bookings from scratch (without a quote) using the **Create Booking** button on the Bookings page. See [Creating a Booking](booking-builder.md) for the full step-by-step guide.
{% endhint %}
