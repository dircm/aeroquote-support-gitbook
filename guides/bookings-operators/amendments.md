# üìù Amendments

The Amendments section provides a complete audit trail of all changes made to a booking. See who changed what and when.

---

## Accessing Amendments

1. Go to **Bookings** from the sidebar
2. Click on a booking
3. Select the **Amendments** tab

---

## What's Tracked

Amendments capture changes to:

| Category | Examples |
|----------|----------|
| **Booking Details** | Customer, reference, dates |
| **Itinerary** | Routes, times, airports |
| **Passengers** | Added, removed, weight changes |
| **Cargo** | Added, removed, weight changes |
| **Crew** | Assignments changed |
| **Status** | Flight status updates |
| **Pricing** | Cost adjustments |
| **Aircraft** | Aircraft changes |

---

## Amendment History View

The amendments list shows:

| Column | Description |
|--------|-------------|
| **Date/Time** | When the change was made |
| **User** | Who made the change |
| **Field** | What was changed |
| **Old Value** | Previous value |
| **New Value** | Updated value |
| **Notes** | Any notes added |

## Why Amendments Matter

### Operational Clarity
- Know what changed and when
- Understand the booking's evolution
- Resolve disputes with clear history

### Compliance
- Audit trail for regulations
- Documentation for insurance
- Record keeping requirements

### Team Communication
- See changes made by colleagues
- Understand current state
- Avoid conflicting updates

---

## Adding Notes to Changes

When making significant changes:

1. Make your change
2. Add a note explaining why
3. Save

The note is stored with the amendment record.

{% hint style="info" %}
Good notes help future-you (and your team) understand why changes were made.
{% endhint %}

---

## Filtering Amendments

For bookings with many changes:
- Filter by date range
- Filter by change type
- Filter by user
- Search for specific changes

---

## Reverting Changes

Amendments are a history log ‚Äî they don't directly support "undo." To reverse a change:

1. View the amendment to see old value
2. Manually update the field back
3. This creates a new amendment record

---

## Amendment Reports

You may be able to:
- Export amendment history
- Include in booking reports
- Print for records

## Tips

{% hint style="success" %}
**Review before departure** ‚Äî Check amendments before a flight to ensure you're aware of any last-minute changes.
{% endhint %}

- Add notes for non-obvious changes
- Review amendments when taking over a booking
- Use amendments to troubleshoot issues
- Keep amendment history for dispute resolution
