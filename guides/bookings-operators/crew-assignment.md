# 👨‍✈️ Crew Assignment

The Crew section lets you assign pilots, co-pilots, and cabin crew to each flight in a booking. Keep track of who's flying what.

---

## Accessing Crew Assignment

1. Go to **Bookings** from the sidebar
2. Click on a booking
3. Navigate to the crew assignment area (often within Itinerary or a dedicated Crew tab)

---

## Prerequisites

Before assigning crew, you need staff members set up:

{% content-ref url="../staff/README.md" %}
[Staff Management](../staff/README.md)
{% endcontent-ref %}

Staff members marked as crew-eligible will appear in the crew selection.

---

## Assigning Crew to a Flight

### Step 1: Select the Flight Leg

Choose which flight leg you're assigning crew to.

### Step 2: Add Crew Members

| Role | Description |
|------|-------------|
| **Pilot in Command (PIC)** | Captain / primary pilot |
| **Second in Command (SIC)** | Co-pilot / first officer |
| **Cabin Crew** | Flight attendants (if applicable) |

### Step 3: Select from Staff

Choose crew members from your staff list. The dropdown shows:
- Staff member name
- Their role/position
- Current availability (if scheduling is used)

### Step 4: Save

Confirm the crew assignment.

---

## Multi-Leg Assignments

For trips with multiple legs, you can:

- **Same crew throughout** — Assign once, applies to all legs
- **Different crew per leg** — Mix and match for complex trips
- **Crew changes** — Handle crew swaps at intermediate stops

---

## Crew on the Dashboard

Assigned crew appears on:
- The booking details
- The Fleet Timeline (shows crew commitments)
- Calendar view

This helps avoid double-booking crew members.

---

## Crew Notifications

Assigned crew members receive:
- Email notification of assignment
- Calendar invite for the flight (synced to their device calendar)
- Access to view booking details
- **Push notifications** via the mobile app for departure and arrival alerts

{% hint style="info" %}
Crew must have the AeroQuote mobile app installed to receive push notifications. Notifications are sent automatically when departures and arrivals are detected by flight tracking or confirmed by crew.
{% endhint %}

---

## Mobile App for Crew

Assigned crew can use the AeroQuote mobile app (iOS and Android) to:
- View upcoming flights with a countdown to departure
- Check in passengers manually or by scanning QR boarding passes
- Confirm departures and arrivals with timestamps
- Update ETAs from the field
- Record flight status even offline — data syncs when connectivity returns

All actions from the mobile app sync to the Bookings Dashboard and your team in real time.

---

## Viewing Crew Across Bookings

To see a crew member's schedule:
1. Go to **Staff**
2. Select the crew member
3. View their assigned flights

Or use the **Fleet Timeline** on the Dashboard to see crew allocations visually.

---

## Changing Crew

If you need to swap crew:

1. Open the booking
2. Go to crew assignment
3. Remove the current crew member
4. Assign the replacement
5. Save changes

{% hint style="info" %}
Crew changes are tracked in the [Amendments](amendments.md) history.
{% endhint %}

---

## Tips

{% hint style="success" %}
**Assign early** — Confirm crew as soon as possible so they can plan accordingly.
{% endhint %}

- Check crew availability before assigning
- Consider duty time limits and rest requirements
- Note any aircraft type ratings needed
- Communicate changes promptly to affected crew
